## Installation
```bash
$ cd /path/to/project
$ cp .env.example .env
$ npm i
```

## Available Scripts
In the project directory, you can run:

### `npm run lint`
Checks code for Code Style issues.

### `npm run build`
Builds the server for production.

### `npm run dev`
Starts local server for development on http://localhost:8080.

### `npm run start`
Runs the server for production.
